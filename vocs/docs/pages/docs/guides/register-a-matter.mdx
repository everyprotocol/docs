# Register a Matter

This guide walks you through the process of registering a matter on the Every Network.

## Before You Begin

Make sure you have:

1. Learned the [concepts of matters](/docs/concepts/elements#matter)
2. A [wallet set up](/docs/guides/set-up-wallets) and funded with tokens

## Register a Matter

To register a matter using the CLI:

```bash
every matter register image.png --account eve --network devent
```

## Access the Matter

a matter can be retrieved through Every Network gateways using its hash.

- `https://every.bz/matter/{hash}`

For example, the matter we just registered can be accessed at: https://every.bz/matter/0x414dc89b6efd259a6f08b900f689649427f4932f04f1c7f0475f381f87544a96.

 See [endpoints](/docs/network/endpoints) for all public gateways.
