Primary interface for kind registration and management


## Functions
### register

Registers a new kind with the specified parameters


```solidity
function register(ElementType[] memory shape, bytes32 code, bytes32 data, uint64[] memory rels)
    external
    returns (uint64);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`shape`|`ElementType[]`|Specification of the elements that define the object's state|
|`code`|`bytes32`|The material hash of the kind's code|
|`data`|`bytes32`|The material hash of the kind's data|
|`rels`|`uint64[]`|List of relations supported by this kind|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`<none>`|`uint64`|The ID of the newly registered kind|


### update

Updates the code and data of an existing kind


```solidity
function update(uint64 id, bytes32 code, bytes32 data) external returns (ObjectMeta memory);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the kind to be updated|
|`code`|`bytes32`|The new material hash representing the kind's code (zero = no change)|
|`data`|`bytes32`|The new material hash representing the kind's data (zero = no change)|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`<none>`|`ObjectMeta`|The updated metadata for the kind|


### update

|`<none>`|`ObjectMeta`|The updated metadata of the kind|

## Events
### KindRegistered
Emitted when a new kind is registered


```solidity
event KindRegistered(
    uint64 id, ObjectMeta meta, ElementType[] shape, bytes32 code, bytes32 data, uint64[] rels, address owner
);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the newly registered kind|
|`meta`|`ObjectMeta`|Metadata of the registered kind|
|`shape`|`ElementType[]`|The element specification of objects belonging to this kind|
|`code`|`bytes32`|The material hash of the kind's code|
|`data`|`bytes32`|The material hash of the kind's data|
|`rels`|`uint64[]`|List of relations supported by this kind|
|`owner`|`address`|The address that owns the registered kind|

### KindUpdated
Emitted when a kind is updated


```solidity
event KindUpdated(uint64 id, ObjectMeta meta, uint64[] rels, bytes32 code, bytes32 data);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the updated kind|
|`meta`|`ObjectMeta`|Updated metadata of the kind|
|`rels`|`uint64[]`|Updated list of relations associated with the kind|
|`code`|`bytes32`|Updated material hash of the kind's code|
|`data`|`bytes32`|Updated material hash of the kind's data|

### KindUpgraded
Emitted when a kind is upgraded to a new revision


```solidity
event KindUpgraded(uint64 id, ObjectMeta meta);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the upgraded kind|
|`meta`|`ObjectMeta`|Updated metadata of the upgraded kind|

### KindTouched
Emitted when a kind's revision is bumped without state change


```solidity
event KindTouched(uint64 id, ObjectMeta meta);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the kind|
|`meta`|`ObjectMeta`|Metadata of the touched kind|

### KindTransferred
Emitted when ownership of a kind is transferred


```solidity
event KindTransferred(uint64 id, address from, address to);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the kind being transferred|
|`from`|`address`|The current owner of the kind|
|`to`|`address`|The new owner of the kind|
