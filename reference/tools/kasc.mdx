Compiler for writing Kind contracts in AssemblyScript.

<Note>
  This is the core binary included in kind-as, used to compile .ts source files into WebAssembly modules that run on the Every Chain. For language reference and development guides, see the [kind-as documentation](https://kindas.previous.network/index.html).
</Note>

## Install

```sh-session
$ npm install -g @everyprotocol/kind-as
```

## Usage

```sh-session
$ npm install -g @everyprotocol/kind-as
```
