# Localnet

* -fa eve
* -a alice
* alice.ss: root
* bob.ss: herald of the anvil universe

## start universe chain

```bash
git clone git@github.com:everyprotocol/localnet.git
cd localnet
```

```bash
docker compose up -d --pull never
```

```bash
docker compose down
```

```bash
rm -rf data
```

## deploy contracts


```bash
git clone git@github.com:everyprotocol/core.git
cd core
```

```bash
C3F=0x5FbDB2315678afecb367f032d93F642f64180aa3 \
SK=0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80 \
forge script script/Deploy.s.sol --sig deployAll \
--libraries contracts/libraries/Snapshots.sol:Snapshots:0x7F26855FA78ab0e511fdd42C75fCf2F7C96074B8 \
--rpc-url http://anvil.localhost \
--broadcast
```

```bash
bash script/verify-contract.sh
```
