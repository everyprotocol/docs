Command line interface for the Every Protocol

## Install

```sh-session
$ npm install -g @everyprotocol/every-cli
```

## Usage

```sh-session
$ every COMMAND
running command...
$ every (--version)
@everyprotocol/every-cli/0.0.3 darwin-arm64 node-v22.6.0
$ every --help [COMMAND]
USAGE
  $ every COMMAND
...
```


## Commands

* [`every autocomplete [SHELL]`](#every-autocomplete-shell)

### `every autocomplete [SHELL]`

Display autocomplete installation instructions.

```
USAGE
  $ every autocomplete [SHELL] [-r]

ARGUMENTS
  SHELL  (zsh|bash|powershell) Shell type

FLAGS
  -r, --refresh-cache  Refresh cache (ignores displaying instructions)

DESCRIPTION
  Display autocomplete installation instructions.

EXAMPLES
  $ every autocomplete

  $ every autocomplete bash

  $ every autocomplete zsh

  $ every autocomplete powershell

  $ every autocomplete --refresh-cache
```

_See code: [@oclif/plugin-autocomplete](https://github.com/oclif/plugin-autocomplete/blob/v3.2.26/src/commands/autocomplete/index.ts)_
