Interface for set registration and management


## Functions
### register

Registers a new set in the registry


```solidity
function register(address law, bytes32 lineage) external returns (uint64);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`law`|`address`|The address of the set contract|
|`lineage`|`bytes32`|The material hash of the set's data|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`<none>`|`uint64`|The ID of the newly registered set|


### update

Updates the metadata and description of a set


```solidity
function update(uint64 id, bytes32 lineage) external returns (ObjectMeta memory);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the set to update|
|`lineage`|`bytes32`|The updated metadata or description hash|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`<none>`|`ObjectMeta`|The updated metadata of the set|


### upgrade

Upgrades object to newer kind/set revisions


```solidity
function upgrade(uint64 id, uint32 kindRev, uint32 setRev) external returns (ObjectMeta memory);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the object|
|`kindRev`|`uint32`|New kind revision (0 = no upgrade)|
|`setRev`|`uint32`|New set revision (0 = no upgrade)|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`<none>`|`ObjectMeta`|Updated metadata after upgrade|


### touch

function touch(uint64 id) external returns (ObjectMeta memory);

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the object|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`<none>`|`ObjectMeta`|Updated metadata after touch|

## Events
### SetRegistered
Emitted when a new set is registered


```solidity
event SetRegistered(uint64 id, ObjectMeta meta, address code, bytes32 data, address owner);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the set|
|`meta`|`ObjectMeta`|The initial metadata of the set|
|`code`|`address`|The address of the set's contract|
|`data`|`bytes32`|The material hash of the set's data|
|`owner`|`address`|The address of the set's owner|

### SetUpdated
Emitted when a set is updated


```solidity
event SetUpdated(uint64 id, ObjectMeta meta, bytes32 data);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the set|
|`meta`|`ObjectMeta`|Updated metadata of the set|
|`data`|`bytes32`|Updated material hash of the set's data|

### SetUpgraded
Emitted when a set is upgraded


```solidity
event SetUpgraded(uint64 id, ObjectMeta meta);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the set|
|`meta`|`ObjectMeta`|The metadata of the set after upgrade|

### SetTouched
Emitted when a set is touched


```solidity
event SetTouched(uint64 id, ObjectMeta meta);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the set|
|`meta`|`ObjectMeta`|The metadata of the set after touch|

### SetTransferred
Emitted when ownership of a set is transferred


```solidity
event SetTransferred(uint64 id, address from, address to);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the set|
|`from`|`address`|The previous owner's address|
|`to`|`address`|The new owner's address|
