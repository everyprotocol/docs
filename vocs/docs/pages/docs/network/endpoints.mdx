# Endpoints

Every Protocol networks can be accessed through RPC and gateway endpoints.

## Mainnet  _(stay tuned)_
Recommended for production use.
- RPC: `https://mainnet.every.network`
- Gateway: `https://every.fm`

## Testnet _(coming soon)_
Used for staging and public testing.
- RPC: `https://testnet.every.network`
- Gateway: `https://every.im`

## Devnet
Experimental development environment.
- RPC: `https://devnet.every.network`
- Gateway: `https://every.gm`

## Localnet
Run locally on your own machine for development and testing.
- RPC: `http://every.localhost`
- Gateway: `http://every.lo`
