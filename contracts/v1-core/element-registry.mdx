Core interface for element registration and management


## Functions
### registerValue

Registers a new value element


```solidity
function registerValue(TokenStandard std, address addr, bytes32 material) external returns (uint64 id);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`std`|`TokenStandard`|Token standard (e.g., ERC20)|
|`addr`|`address`|Token contract address|
|`material`|`bytes32`|Token material hash|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the newly registered value element|

## Events
### ValueRegistered
Emitted when a new value element is registered


```solidity
event ValueRegistered(uint64 id, TokenStandard std, uint8 decimals, address addr, bytes32 material, address admin);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the registered value element|
|`std`|`TokenStandard`|The token standard|
|`decimals`|`uint8`|The number of decimals for the token|
|`addr`|`address`|The token contract address|
|`material`|`bytes32`|The token material hash|
|`admin`|`address`|The address of the admin who registered the value|

### ValueUpdated
Emitted when a value element is updated


```solidity
event ValueUpdated(uint64 id, bytes32 material);
```

**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`id`|`uint64`|The ID of the updated value element|
|`material`|`bytes32`|The new material hash|
